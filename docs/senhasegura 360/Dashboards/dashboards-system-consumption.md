These dashboards allow administrators to monitor consumption indicators, providing a consolidated view of the product's key resources.

:::(Info) (Info)
The data in these dashboards is consolidated once a day, so there may be discrepancies from analytical reports.
:::

Through the menu , you have access to the following items:

### General

|  |  |
| --- | --- |
|  | It displays the day when viewing the dashboard. |
|  |It displays the total number of users with access to the system. |
|  |It displays the total number of users with access to PAM.|
|  |It displays the total number of users with access to GO. |
|  |It displays the total number of users with access to Certificate Manager.|
|  | It displays the total number of users with access to Domum.|
|  |It displays the number of devices in the system, categorized by type. |
|  |It displays the total number of applications and secrets registered in DSM. |
|  |It displays the total number of workstations registered in GO. |
|  |It displays the total number of applications registered in A2A. |
|  |It displays the total number of active certificates in the system. |
|  |It displays the total number of users with access to MySafe. Exclusive users have the permission "MySafe.Secret.List" and do not have the permission "PAM.PrivilegedAccounts.Credentials.List". |
|  |It displays the total number of members in the cluster.|
|  |It displays the total number of recordings in the cluster environment. |
|  |It displays a progression graph showing the number of active devices in the system per month. |
